# HR Attrition Analysis (Excel Project)

## Overview

* This project focuses on analyzing employee attrition trends using HR data
* It examines factors such as age, department, job role, salary, and satisfaction levels
* The objective is to identify turnover patterns and support HR retention strategies

## Dataset

* File name: HRattrition.xlsx
* Contains employee demographics, job roles, compensation details, satisfaction scores, and attrition status
* Used to identify key drivers of employee turnover and aid HR decision-making

## Objectives

* Identify major factors influencing employee attrition
* Analyze attrition patterns across departments, roles, and demographics
* Provide insights to improve workforce retention strategies

## Tools and Techniques

* Excel functions for data cleaning, formulas, and conditional formatting
* Pivot tables to summarize attrition by department, job role, and demographics
* Charts and graphs including bar charts, pie charts, and line charts
* Interactive Excel dashboard for visual analysis and insights

## Key Insights

* Higher attrition observed in specific departments such as Sales
* Employees with lower satisfaction levels show increased turnover
* Younger employees tend to leave within the first two years

## Usage

* Open the HRattrition.xlsx file in Microsoft Excel
* Review pivot tables and charts to explore attrition patterns
* Use the dashboard to understand insights and recommendations
